# FSDHomeWork
Semester 4 FSD homework! AI code.

## Apps

### `uvscreen`

Express + EJS app powered by the OpenUV API. It now stores recent UV lookups in MongoDB and shows the latest saved searches on the home page.

Environment variables:

- `OPENUV_API_KEY`
- `PORT`
- `MONGO_URI`

### `contact-manager`

Full-stack contact manager with:

- React frontend in `contact-manager/frontend`
- Express + MongoDB backend in `contact-manager/backend`

Backend environment variables:

- `PORT`
- `MONGO_URI`

### `student-feedback-system`

Express + MongoDB backend with a static frontend for collecting and managing course feedback.

Environment variables:

- `PORT`
- `MONGO_URI`
