# Contact Manager

Full-stack contact manager with a React frontend and an Express + MongoDB backend.

## Run locally

### Backend

1. Install dependencies in `contact-manager/backend`
2. Set `MONGO_URI` if needed
3. Run `npm run dev`

### Frontend

1. Install dependencies in `contact-manager/frontend`
2. Optionally set `VITE_API_BASE_URL`
3. Run `npm run dev`
